# MouseFinder

A small Windows tool that centers your mouse cursor on the current monitor when you press **Ctrl+Shift**, with undo/redo via F8.

## Hotkeys
- **Ctrl + Shift** — center mouse on current monitor (saves previous position)
- **F8** — undo last move
- **Shift + F8** — redo
- **Ctrl + Shift + 7** — quit
- **Esc** — emergency quit
